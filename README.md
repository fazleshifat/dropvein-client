# DropVein - Blood Donation Platform (Frontend)

Welcome to the frontend of **DropVein**, a modern blood donation web application that connects donors and recipients efficiently. This React-based frontend is designed with user-friendly UI, responsiveness, and clean user experience in mind.

## ✨ Features

- 🔐 **Authentication System**: Users can register, log in, and access dashboard roles based on Firebase Auth.
- 👤 **Role-Based Dashboard**: Separate views for donors, admins, and users.
- 📝 **Create & Manage Donation Requests**: Donors can post requests and update them from the dashboard.
- 🗃️ **Donation Request Listing**: View all available requests in real time.
- 🔍 **Request Filtering**: Filter donation requests by blood group, district, and urgency.
- 🕵️ **Admin Control Panel**: Admin can manage users, ban/unban accounts, and view system stats.
- 💬 **Contact Section**: Built-in responsive contact form for users to reach the admin.
- 📱 **Responsive Design**: Mobile-first layout using Tailwind CSS and DaisyUI.
- 🌙 **Dark Mode Support**: Seamless toggle between light and dark themes.
- 🚀 **Smooth Animation**: Subtle animations using `react-awesome-reveal` and modern UI components.

## 🛠 Tech Stack

- React.js
- React Router
- Firebase Auth
- DaisyUI + Tailwind CSS
- React Hook Form
- React Query
- SweetAlert2
